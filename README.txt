The makefile is configured to run in multithreaded mode with openmp by default.
To make the application run in singlethreaded mode, remove the "-fopenmp" flag from the makefile.

When run correctly, the program will output tokens and their frequency from the text file "theHeadband.txt" to the text file "OUTPUT.txt"
Output tokens will be ordered by frequency, descending.

TO CHANGE THE INPUT FILE:
Replace the text in "theHeadband.txt"
